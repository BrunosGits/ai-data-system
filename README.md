# AI Data System

## Overview
This project is an evolving AI system that starts simple and grows into a full AI + data + backend platform.

---

## Current Phase: Phase 1 — Minimal AI API

### Goal
Build a simple API endpoint that:
- Receives a question
- Sends it to an LLM
- Returns a response

### Scope (Strict)
- FastAPI backend
- One endpoint: POST /ask
- No database
- No frontend
- No agents
